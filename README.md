# TYPO3 Extension ayacoo_soundcloud

## 1 Features

* PeerTube videos can be created as a file in the TYPO3 file list


## 2 Usage

### 2.1 Installation

#### Installation using Composer

The recommended way to install the extension is using Composer.

Run the following command within your [Composer][1] based TYPO3 project:

```
composer require cri/cri_peertube
```

### 2.2 Hints

Enter your Peertube Server in the Extension Settings.
Manage your Cookies and Privacy Settings
